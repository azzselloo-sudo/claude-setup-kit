# 클로드코드 셋업 키트

현지의 실제 셋업과 일하는 방식을 그대로 물려주는 키트입니다.
이걸 따라 하면 클로드코드가 **"말 잘 듣고 일 잘하는 상태"** 가 됩니다.

> 전제 : 클로드코드는 이미 깔려 있다고 봅니다. (아직이면 클로드코드부터 설치)

---

## 빠른 시작 (3단계)

받는 사람은 이 3단계만 하면 됩니다.

1. **스킬 깔기** → `02-skills-and-mcp.md` 의 명령어를 클로드코드에 한 줄씩 입력
2. **규칙서 넣기** → `CLAUDE.md` 를 본인 작업 폴더 맨 위에 복사 (또는 클로드에게 "이 CLAUDE.md 내 일에 맞게 고쳐줘")
3. **폴더 만들기** → 클로드에게 "`01-folder-structure.md` 구조대로 폴더 만들어줘"

> 다운로드 : 초록 **Code** 버튼 → Download ZIP, 또는 클로드코드에 "이 깃헙 받아서 내 폴더에 풀어줘 + (레포 주소)"

---

## 구성

**A. 셋업 (먼저 이대로 깔기)**

| 파일 | 내용 |
|---|---|
| `CLAUDE.md` | 클로드가 매번 읽는 규칙서 템플릿 |
| `01-folder-structure.md` | 폴더 구조 (숫자 먼저 정리법) |
| `02-skills-and-mcp.md` | 스킬·플러그인·MCP 설치 (실제 명령어) |
| `03-tool-setup.md` | 도구별 셋업·활용 노하우 |

**B. 일하는 방식 (셋업 후 읽으면 실력이 붙는 부분)**

| 파일 | 내용 |
|---|---|
| `04-work-style.md` | 작업 규칙 전부 (워크플로우·글쓰기·디자인·콘텐츠) |
| `05-build-methodology.md` | AI 빌드 7단계 + 코딩 컨벤션 + 자동화 + 프롬프트 |
| `06-client-playbooks.md` | 클라이언트 작업 표준 절차·플레이북·검증 |
| `07-content-growth.md` | 콘텐츠 공식 + 그로스 퍼널 + 서비스화 |
